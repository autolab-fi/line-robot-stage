Комплекс для дистанционного обучения "Робот линии".


Комплекс состоит из колёсного робота и полигона.


Управление роботом осуществляется с помощью микроконтроллера ESP-WROOM-32. 
Программу в контроллер можно загружать дистанционно через специальную страницу.
В текущей комплектации на роботе установлены датчики линии и светодиоды-фары. Предусмотрена возможность крепления и подключения дополнительных модулей.
Робот может работать в автономном режиме: на борту установлен аккумулятор большой емкости.
Предусмотрена возможность самостоятельной подзарядки на станции с автоматическим позиционированием робота.

Полигон включает в себя поле с рисунком, по которому движется робот, ограждение, крепежную раму и одноплатный компьютер Raspberry Pi с камерой, с которого ведётся видеотрансляция.
С помощью камеры можно отслеживать положение робота. На верху робота закреплен специальный тег.

Более подробное описание робота и инструкцию по сборке вы можете найти в файле: "Название_файла"
